Major Pieces of the Web

The web is made up of computers that communicate with each other over a public network called the internet. When you go to a web page like www.google.com, an HTTP request is sent to a server. The server retrieves the requested HTML document and sends it back to your computer to be interpreted by your web browser and displayed. This video does a great job of explaining it.

HTML

HTML stands for Hypertext Mark up Language. The world wide web is a collection of HTML documents. HTML is like the glue that holds everything together and is the basis for all documents on the web. Watch this video .

Tags and Elements

An HTML mark up is made up of tags and elements . Tags are keywords within a web page that define how the browser displays the content. Most tags have two parts, and opening tag and a closing tag. Elements refers to everything within a set of opening and closing tags.

Inline and Block

HTML elements are displayed either as inline or block. A block element breaks the content up vertically whereas an inline element flows within the content. This video does a great job of explaining it.
